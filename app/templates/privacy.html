Privacy Policy, Security Measures & Terms of Service

Privacy Policy

This privacy policy outlines how we handle your data when using MessagePilot in compliance with the General Data Protection Regulation (GDPR) and other applicable data protection laws.

Legal Basis for Processing

  • Legitimate Interest (Article 6(1)(f)): Processing contact data to facilitate communication for legitimate business purposes
  • Consent (Article 6(1)(a)): You provide explicit consent by uploading and using contact data
  • Contract Performance (Article 6(1)(b)): Processing necessary to provide the messaging service you requested

Data Controller Information

  • Data Controller: The organization using this tool (you or your organization)
  • Data Processor: SGP Digital Solutions (provides the technical processing service)
  • Contact: Users are responsible for providing contact details for data subject requests

Data Subject Rights (GDPR Articles 15-22)

  • Right of Access: Since no data is stored, there is no persistent data to access
  • Right to Rectification: Data can be corrected in the source file before upload
  • Right to Erasure: All data is automatically deleted after processing
  • Right to Restrict Processing: Users can stop processing at any time
  • Right to Data Portability: Original data remains with the user
  • Right to Object: Users can object to processing by not using the service

Data Collection & Processing

  • We process your spreadsheet data locally on your device/server
  • No data is stored permanently - all files are processed in memory
  • Uploaded files are automatically deleted after processing
  • We do not track, store, or analyze usage patterns
  • No cookies are used except those essential for basic functionality
  • User preferences are stored locally in your browser only

Privacy-Preserving WhatsApp Integration

  • Clipboard Method (Default): Personal data is copied securely to your clipboard, only phone numbers are shared in URLs
  • Preview Before Send: Review and edit messages before they are sent
  • No URL Data Exposure: Names, locations, and personal information are not included in browser history or server logs
  • Privacy-Only Methods: All available methods prioritize privacy protection
  • Local Storage: All preferences are stored locally on your device, not on our servers

Data Usage

  • Contact information is used solely for generating WhatsApp message links
  • No data is shared with third parties
  • No analytics or tracking services are implemented
  • Personal data is never exposed in URLs - all methods are privacy-preserving

Security Measures

Technical Security

  • All phone numbers are masked in the user interface
  • Secure URL handling prevents phone numbers from appearing in browser history
  • Implementation of strict Content Security Policy (CSP) headers
  • Prevention of clickjacking through X-Frame-Options
  • MIME-type sniffing prevention via X-Content-Type-Options
  • Referrer Policy controls to prevent data leakage

Data Protection

  • PII (Personally Identifiable Information) filtering in logs
  • Secure file handling and validation
  • Input sanitization and validation
  • Memory-only processing - no persistent storage

Terms of Service

Acceptable Use

  • This tool must be used in compliance with WhatsApp's Terms of Service
  • The tool generates standard wa.me links for manual messaging only
  • Automated messaging or scraping is strictly prohibited
  • Users must have proper consent to message the contacts
  • Users are responsible for compliance with local privacy laws

Limitations

  • The tool is provided "as is" without warranties
  • We are not responsible for misuse or unauthorized use
  • We reserve the right to modify or terminate service
  • Users must not attempt to circumvent security measures

WhatsApp Compliance

  • This tool uses official WhatsApp click-to-chat links (wa.me)
  • No automation or bot functionality is provided
  • Messages are sent manually by the user
  • Users must comply with WhatsApp's messaging limits
